# 🚀 AR Learning v5.0 - Quick Start Guide


## 🔬 Physics Simulations

Each concept type uses real physics:

### Molecular Dynamics
- **Physics**: Coulomb repulsion between charged particles
- **Equation**: F = k|q₁q₂|/r²
- **Visual**: Electrons orbit nucleus, repel each other
- **Realism**: ⭐⭐⭐⭐⭐ Expert

### Orbital Mechanics
- **Physics**: Newton's law of universal gravitation
- **Equation**: v = √(GM/r)
- **Visual**: Planets orbit star with correct speeds
- **Realism**: ⭐⭐⭐⭐⭐ Expert

### Particle Systems
- **Physics**: Emitter + velocity + air resistance + lifetime
- **Equations**: a = F/m, v_new = v_old + a*dt, damping
- **Visual**: Flow from source, fade with age
- **Realism**: ⭐⭐⭐⭐ Advanced

### Cellular Automata
- **Physics**: Conway's Game of Life rules
- **Equations**: Birth if 3 neighbors, survive if 2-3 neighbors
- **Visual**: Patterns emerge and evolve
- **Realism**: ⭐⭐⭐⭐ Advanced

### Wave Simulation
- **Physics**: 2D wave equation (∂²u/∂t² = c²∇²u)
- **Equations**: Laplacian-based propagation
- **Visual**: Waves ripple and interfere
- **Realism**: ⭐⭐⭐⭐⭐ Expert

## ⚡ Performance Tips

### Maximize Smoothness
- Close other browser tabs
- Use modern browser (Chrome 90+, Firefox 88+, Safari 15+)
- Disable unnecessary browser extensions
- Ensure hardware acceleration enabled

### Check Performance
- Should see ~60 FPS
- Smooth animations (no stuttering)
- Quick response to gestures (<100ms)
- Voice recognition works (if available)

### If Performance Drops
- Reduce screen resolution
- Close background apps
- Check internet connection (for Wikipedia)
- Clear browser cache

## 🆘 Troubleshooting

### No 3D Appears
```
✓ Wait 2-3 seconds for Wikipedia to load
✓ Check browser console (F12) for errors
✓ Try different lesson
✓ Refresh page (Ctrl+R)
✓ Clear cache (Ctrl+Shift+Del)
```

### Wikipedia Not Loading
```
✓ Check internet connection
✓ Try again (API may be slow)
✓ System uses fallback if fails
✓ Check browser allows CORS requests
```

### Voice Doesn't Work
```
✓ Click 🎤 button (may need permission)
✓ Allow microphone access when prompted
✓ Check internet (needed for recognition)
✓ Use clear, loud voice
✓ Try different browser
```

### Touch Gestures Not Working
```
✓ Ensure two-finger swipe (if needed)
✓ Try from center of screen
✓ Check browser supports touch events
✓ Try game controller instead
```

### Game Controller Not Detected
```
✓ Plug in controller before loading page
✓ Press a button to activate
✓ Check browser supports Gamepad API
✓ Try refreshing page
```
