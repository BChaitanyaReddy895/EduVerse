## 🚀 HOW IT WORKS

### Automatic Flow

```
1. User views concept with engineering topic
   ↓
2. System analyzes topic keywords
   ↓
3. EngineeringConceptDetector identifies concept type
   ↓
4. If engineering concept found:
   - Routes to EngineeringGradeVisualization
   - Applies realistic lighting setup
   - Creates CAD-quality geometry
   - Sets up physics-based animations
   
   If NOT engineering concept:
   - Uses simple basic visualization
   - Fast performance on mobile
   
5. Visualization rendered in 3D canvas
6. Metadata displayed in info panel:
   - Engineering keywords
   - Technical specifications
   - Animation description
   - Complexity level
```

### Visualization Modes

#### AUTO MODE (Default)
- Intelligent detection
- Switches automatically between simple and engineering
- Best for all concepts

#### ENGINEERING MODE
- Always use realistic visualizations
- Even if detected as non-engineering, try engineering style
- Higher quality, more demanding (desktop recommended)

#### SIMPLE MODE
- Always use basic visualizations
- Fastest performance
- Best for mobile devices
